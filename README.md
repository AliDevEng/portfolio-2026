## Getting Started

```bash
# Clone the repo
git clone https://github.com/AliDevEng/Portfolio-V1.git
cd Portfolio-V1

# Install dependencies
npm install

# Run dev server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it.
